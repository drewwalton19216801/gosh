gosh

A basic command-line shell written in Go with sh-style scripting support

Designed for macOS compatibility. Gosh, it's good to have a reliable shell!

Get Started View Commands
gosh
$ ./gosh
gosh> echo "Hello, World!"

then> echo "Go project detected"
then> fi

Features

🚀

Multiple Execution Modes

Interactive mode, command mode, and script mode for maximum flexibility

🔗

Unix Pipes & Redirection

Full support for pipes, input/output redirection, and command chaining

📝

Shell Scripting

Write powerful shell scripts with functions, control structures (if/case), variable expansion, and command substitution

⚡

Built-in Commands

Comprehensive set of built-in commands for common shell operations

🌟

Modern Features

Command history, aliases, tab completion, and background execution

🍎

macOS Optimized

Designed and tested specifically for macOS with cross-platform compatibility

Getting Started

1

Build gosh

go build -o gosh
2

Run Interactive Mode

./gosh

Gosh, that was easy!

3

Try Command Mode

./gosh -c "echo 'Hello World'"
4

Execute Scripts

./gosh script.sh

By gosh, scripting has never been simpler!
